Indian Museum Artefact Digitisation System

A JavaFX and MySQL based desktop application developed for managing and exploring museum artefacts.

Features:
 Administrator Login
 Add Artefacts
 Update Artefacts
 Delete Artefacts
 Search and Filter Artefacts
 Museum-themed User Interface
 MySQL Database Integration
Technologies Used:
 Java
 JavaFX
 MySQL
 JDBC
 Git & GitHub
